## 6. 설비 보완 계획 (경량화 전략)

> **기본 방침**: 이 과제의 핵심은 설비 구축이 아니라 **"통합운영 및 거래 기술"의 개발과 실증**. 기존 레거시 자산을 최대 활용하고, 신규 설비는 RFP 요건 충족에 필요한 **저장 설비만 최소 구축**하여 핵심 R&D에 예산을 집중.

### 6-1. 경량화 근거

| 관점                | 설명                                                                                  |
| ------------------- | ------------------------------------------------------------------------------------- |
| **전력 흡수** | 기존 히트펌프만으로 12.4MW → 10MW 요건 이미 초과, 수전해 증설 불필요                 |
| **P2X 종류**  | 기존 4종(P2H, P2C, P2G, P2M) 유지 → 3종 이상 충족                                    |
| **저장 핵심** | TES가 MWh당 가장 저렴 (ESS 대비 1/10), 지역난방 인프라 직결 → TES 중심으로 저장 해결 |
| **예산 배분** | 설비 70~100억 → R&D 10~40억 (X) ↔ 설비 25~35억 → **R&D 75~85억 (O)**             |
| **평가 전략** | 장비 비중 과다 시 "시설구축 과제" 지적 가능, R&D 비중 높여야 기술개발 과제로 인정     |

### 6-2. 신규 구축 설비 (최소 구성)

| 설비                   | 사양                     | 역할                                         | 예상 비용 (억원) |
| ---------------------- | ------------------------ | -------------------------------------------- | ---------------: |
| **TES (축열조)** | **45 MWh급**       | 열에너지 저장 (저비용 대용량), 열부하 유연화 |            12~15 |
| **ESS**          | **0.5 MW / 5 MWh** | 전력시장 대응, 피크 셰이빙                   |            12~18 |
| **신규 합계**    |                          |                                              |  **25~35** |

| 설비          | 조치                                   | 사유                                         |
| ------------- | -------------------------------------- | -------------------------------------------- |
| 수전해 (P2G)  | **기존 0.7MW 유지** (증설 안 함) | P2G 종류 인정에 충분, 증설 비용 20~30억 절감 |
| 수소 저장탱크 | **기존 규모 활용**               | 기존 수소생산 인프라 연계, 5~10억 절감       |

### 6-3. 경량화 후 목표 구성

```
전력 흡수 용량 (≥ 10MW)
├── 히트펌프 P2H/P2C : ~12.4 MW (기존)  ← 이것만으로 10MW 초과
├── 수전해 P2G       : ~0.7 MW (기존 유지)
├── ESS 충전         : ~0.5 MW (신규, 소규모)
├── EV 충전 P2M      : ~0.5 MW (기존)
└── 합계              : ~14.1 MW ✓

에너지 저장 용량 (≥ 50MWh)
├── TES/cTES         : ~45 MWh (신규, 저비용 대용량)  ← 축열이 가장 경제적
├── ESS              : ~5 MWh (신규, 전체의 10% → 20% 이하 ✓)
└── 합계              : ~50 MWh ✓
```

### 6-4. 예산 배분 비교

## 1. 실제 도심형 복합 수요처 기반 실증

> 대부분의 P2X 실증 과제가 산업단지나 격리된 마이크로그리드에서 수행되는 것과 차별화

| 수요처                         | 부하 특성                         | 유연성 가치                   |
| ------------------------------ | --------------------------------- | ----------------------------- |
| **주거 (26.5만 세대)**   | 아침·저녁 피크, 계절별 난방 수요 | 열부하 시간이동(TES)          |
| **산업 (데이터센터 등)** | 24시간 베이스로드, 냉방 대량 소비 | 대용량 전력 수요 DR 참여 가능 |
| **상업시설**             | 주간 피크, 냉난방 동시 수요       | P2H/P2C 전환 운영             |

**부하 유형이 다양할수록 최적 운영 알고리즘의 기술적 가치가 높아집니다.** 단일 수요처 대비 훨씬 복잡한 최적화 문제를 풀어야 하므로, R&D 성과의 기술 수준이 높게 평가됩니다.

## 3. TES 중심 저장의 경제적 우위

| 저장 방식               | MWh당 비용 (억원) | 비고                                   |
| ----------------------- | ----------------- | -------------------------------------- |
| 리튬이온 ESS            | 3~4               | 고비용, 안전관리 부담                  |
| **온수 축열 TES** | **0.3~0.5** | **ESS 대비 1/10, 지역난방 직결** |
| 수소 저장               | 5~8               | 최고비용                               |

**지역난방 인프라와 TES가 직결**되기 때문에:

* 별도 열배관 구축 비용 없음
* 축열한 에너지를 즉시 26.5만 세대에 공급 가능
* 동·하·간절기 모두 열수요가 있어 3,000시간 운전 자연 달성

---

## 4. 다중 수익 모델 실증 가능

```
복합에너지시스템
├── 전력시장 참여    : 예비력·실시간시장 입찰 (ESS + DR)
├── 열 판매         : 지역난방 요금 수입 (TES → 26.5만 세대)
├── 수소 판매       : 충전소 수익 (P2G → P2M)
├── 전기차 충전     : 충전 서비스 수익
└── 탄소 크레딧     : 온실가스 감축량 거래
```

**단일 에너지 시스템에서는 불가능한 복합 거래모델**을 실증할 수 있습니다. 이것이 RFP가 요구하는 "에너지 거래모델 개발 및 실증"의 핵심입니다.
